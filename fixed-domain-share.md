# Fixed domain share

Your fixed public URL is:

```text
https://study.luoge.us.ci
```

Use:

```bat
start-fixed-tunnel.bat
```

This starts:

- the local AI bridge
- the local website server
- the named Cloudflare Tunnel

To stop everything:

```bat
stop-fixed-tunnel.bat
```

Notes:

- The URL stays the same.
- Your computer still needs to stay online.
- If the computer restarts, run `start-fixed-tunnel.bat` again.
